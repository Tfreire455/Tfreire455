### AI, LLMs & AUTOMATION

</div>

<br>

<table>
<tr>
<td width="50%" valign="top">

<br>

**AUTONOMOUS BOTS**

Sales engine running 24/7 on WhatsApp groups via Baileys. Campaign scheduling, smart media delivery, humanized delays and anti-spam flags stored in PostgreSQL.

Full pipeline: Google Sheets to database sync, AI-generated copy, WhatsApp dispatch, real-time Socket.IO dashboard.

![Baileys](https://img.shields.io/badge/Baileys-25D366?style=flat-square&logo=whatsapp&logoColor=white)
![Node.js](https://img.shields.io/badge/Node.js-339933?style=flat-square&logo=nodedotjs&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-4169E1?style=flat-square&logo=postgresql&logoColor=white)
![Socket.IO](https://img.shields.io/badge/Socket.IO-010101?style=flat-square&logo=socketdotio&logoColor=white)

<br>

</td>
<td width="50%" valign="top">

<br>

**LLMs & PROMPT ENGINEERING**

OpenAI API integration for dynamic per-product copywriting. The model receives product name, price, collection and stock level, then generates unique captions with scarcity, urgency and desire triggers.

Three strategic modes -- Normal (elegant), Blitz (urgency) and Collection (storytelling) -- each with a configurable AI personality via admin dashboard.

![OpenAI](https://img.shields.io/badge/GPT--4-412991?style=flat-square&logo=openai&logoColor=white)
![Python](https://img.shields.io/badge/Python-3776AB?style=flat-square&logo=python&logoColor=white)
![LangChain](https://img.shields.io/badge/LangChain-1C3C3C?style=flat-square&logo=langchain&logoColor=white)

<br>

</td>
</tr>

<td width="50%" valign="top">

<br>

**SaaS AUTOMATION**

End-to-end systems that read spreadsheets, sync databases, generate content via LLM and fire campaigns with zero human intervention. Includes CRM layer with lead capture, VIP birthday detection and automatic coupon dispatch through scheduled cron jobs.

![Sheets](https://img.shields.io/badge/Sheets_API-34A853?style=flat-square&logo=googlesheets&logoColor=white)

**RESILIENCE ENGINEERING**

Fault-tolerant architectures designed for month-long uptime: automatic retry with exponential backoff, cascading image fallbacks across Google Drive endpoints, cancellable sleep tokens for infinite loops, and memory-leak prevention patterns.
