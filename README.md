## `AI, LLMs & AUTOMATION`

### `AUTONOMOUS BOTS`

* Sales engine running **24/7** on WhatsApp groups via **Baileys**
* Campaign scheduling, smart media delivery, humanized delays
* Anti-spam flags + history stored in **PostgreSQL**
* Full pipeline: **Google Sheets → DB sync → AI copy → WhatsApp dispatch → Socket.IO dashboard**

### `LLMs & PROMPT ENGINEERING`

* OpenAI API integration for dynamic per-product copywriting
* Inputs: product name, price, collection, stock level
* Outputs: unique captions with scarcity/urgency/desire triggers
* Modes: `Normal (elegant)` • `Blitz (urgency)` • `Collection (storytelling)`
* Configurable AI personality via admin dashboard

### `SaaS AUTOMATION`

* End-to-end systems: read spreadsheets → sync DB → generate content → fire campaigns with **zero manual work**
* CRM layer: lead capture, VIP birthday detection, automatic coupon dispatch via cron

### `RESILIENCE ENGINEERING`

* Automatic retry with exponential backoff
* Cascading media fallbacks (multi-endpoint)
* Cancellable sleep tokens for infinite loops
* Memory-leak prevention patterns + stability-first architecture
